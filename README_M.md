# MySchool (माझी शाळा)

MySchool (माझी शाळा) हे शिक्षकांसाठी डिझाइन केलेले एक सर्वसमावेशक अँड्रॉइड अॅप्लिकेशन आहे. हे शालेय डेटा, विद्यार्थ्यांच्या नोंदी, शैक्षणिक मूल्यमापन आणि प्रगती पुस्तक (Report Card) तयार करण्यासाठी उपयुक्त आहे. हे अॅप इंग्रजी आणि मराठी दोन्ही भाषांमध्ये उपलब्ध असून, Firebase द्वारे क्लाउड सिंक्रोनायझेशनची सुविधा देते.

## वैशिष्ट्ये आणि पृष्ठे (Features & Pages)

### 1. डॅशबोर्ड आणि मुख्यपृष्ठ (Dashboards & Home)
- **आकडेवारी डॅशबोर्ड (`StatsDashboardFragment`)**: वर्ग, विद्यार्थी आणि एकूण प्रगतीची सांख्यिकीय माहिती.
- **अतिरिक्त डॅशबोर्ड (`ExtraDashboardFragment`)**: शिक्षकांसाठी अतिरिक्त सारांश आणि जलद कृती.
- **मुख्यपृष्ठ (Home Activity)**: विविध मॉड्यूल्समध्ये सहजपणे नेव्हिगेट करण्यासाठी मुख्य केंद्र.

### 2. शाळा आणि वर्ग व्यवस्थापन (School & Class Setup)
- **वर्ग आणि तुकडी व्यवस्थापन (`ClassDivListFragment`)**: नवीन वर्ग आणि तुकड्या तयार करणे आणि व्यवस्थापित करणे.
- **विषय व्यवस्थापन (`SubjectsFragment`)**: विविध वर्गांसाठी विषय जोडणे आणि व्यवस्थापित करणे.
- **गुणभार निश्चिती (`DeclareWeightageFragment`)**: विषयांसाठी विशिष्ट गुणदान योजना (तोंडी, प्रात्यक्षिक, लेखी इ.) आणि गुणभार (Weightage) निश्चित करणे.

### 3. विद्यार्थी व्यवस्थापन (Student Management)
- **विद्यार्थी यादी (`StudentListFragment`)**: नवीन विद्यार्थ्यांची नोंदणी करणे, माहिती अद्ययावित (update) करणे आणि वर्ग/तुकडीनुसार विद्यार्थ्यांची संपूर्ण यादी पाहणे.

### 4. शैक्षणिक मूल्यमापन आणि गुणदान (Academic Evaluation)
- **आकारिक आणि संकलित मूल्यमापन (`FormativeSummativeFragment`)**: विविध निकषांनुसार (तोंडी, प्रात्यक्षिक, प्रकल्प, लेखी) गुण भरण्यासाठी एक डायनॅमिक इंटरफेस, ज्यामध्ये आपोआप श्रेणी (Grade) काढली जाते.
- **वर्णनात्मक नोंदी (`DescriptiveEntriesFragment`)**: विद्यार्थ्यांसाठी विशेष वर्णनात्मक नोंदी, जमेच्या बाजू आणि सुधारणेस वाव असलेल्या गोष्टी नोंदवण्याची सुविधा.
- **उपस्थिती नोंद (`AttendanceFragment`)**: विद्यार्थ्यांची दैनंदिन किंवा ठराविक कालावधीची उपस्थिती नोंदवणे.

### 5. अहवाल आणि प्रगती पुस्तक (Report Generation)
- **अहवाल छपाई (`ReportPrintingFragment`, `ReportsFragment`)**: पीडीएफ (PDF) स्वरूपात सर्वसमावेशक प्रगती पुस्तके आणि शैक्षणिक अहवाल तयार करणे.
- **प्रिंट सेटिंग्ज (`InfoPrintSettingFragment`)**: अहवालांवर शाळेची माहिती, लोगो आणि मांडणी कस्टमाइझ करणे.

### 6. सबस्क्रिप्शन आणि प्रोफाइल (Subscriptions & Profile)
- **प्रोफाइल व्यवस्थापन (`ProfileFragment`)**: शिक्षकांची स्वतःची माहिती अद्ययावित करणे.
- **सबस्क्रिप्शन आणि इतिहास (`SubscriptionBottomSheet`, `SubscriptionHistoryActivity`)**: अॅपचे सबस्क्रिप्शन व्यवस्थापित करणे आणि मागील व्यवहारांचा इतिहास पाहणे.
- **सेटिंग्ज (`SettingsFragment`)**: भाषा बदलणे (इंग्रजी/मराठी), थीम आणि इतर अॅप प्राधान्ये व्यवस्थापित करणे.

### 7. अतिरिक्त साधने (Additional Tools)
- **OCR (Optical Character Recognition)**: CameraX आणि Google ML Kit वापरून कागदपत्रे स्कॅन करून त्यातील मजकूर (Text) ओळखण्याची सुविधा.
- **अतिरिक्त मेनू (`ExtraMenusFragment`)**: उपयुक्त साधने आणि मदत विभागापर्यंत पोहोचण्यासाठी.

## तंत्रज्ञान (Tech Stack)
- **भाषा (Language)**: Java 17
- **UI आर्किटेक्चर**: Activities & Fragments, ViewBinding, Navigation Component
- **बॅकएंड आणि डेटाबेस**: Firebase Authentication, Firestore, Storage
- **लायब्ररी (Libraries)**: iText PDF, CameraX & ML Kit OCR, Glide

## प्रकल्प कसा चालवावा (Running the Project)
1. रिपॉझिटरी क्लोन करा आणि Android Studio मध्ये उघडा.
2. Firebase सेवांसाठी `app/` डिरेक्टरीमध्ये `google-services.json` फाइल जोडल्याची खात्री करा.
3. Gradle फाइल्स सोबत प्रकल्प सिंक (Sync) करा.
4. API 24 किंवा त्यापुढील (Target API 35) अँड्रॉइड डिव्हाइसवर किंवा एमुलेटरवर अॅप बिल्ड आणि रन करा.
